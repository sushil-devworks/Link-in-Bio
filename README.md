# 🌐 Link in Bio - Sushil Devworks

A beautiful, modern, and fully responsive **Link in Bio** page that centralizes all my social media handles, major projects, and developer support links in one place. Designed with a mobile-first approach.

## ✨ Features

* **⚡ Minimal & Modern UI:** A clean, eye-catching interface tailored for a sleek user experience.
* **📱 Fully Responsive:** Optimized flawlessly for mobile devices, tablets, and desktops.
* **🔗 Unified Links:** Quick access to YouTube, GitHub, Instagram, WhatsApp, and Facebook.
* **🚀 Project Showcase:** Direct links to live featured projects like **Listy** and **StudyZip**.
* **📲 Native Share Option:** Includes a dynamic 'Share Me' button for easy link sharing.
* **❤️ UPI Support Integration:** Features a 'Support Developer' button using a direct UPI intent link.

## 🛠️ Tech Stack

* **HTML5** - For semantic structure and content.
* **CSS3** - For custom styling, layout layout, and responsiveness.
* **Font Awesome (v6.4.0)** - For clean and scalable social and utility icons.
* **JavaScript** - For handling sharing functionality and interactions.
